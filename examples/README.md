# Zen Language Examples

This directory contains example programs demonstrating various features of the Zen programming language.

## Basic Examples

- **hello.zen** - Simple "Hello, World!" program
- **simple_int.zen** - Basic integer operations
- **arithmetic.zen** - Arithmetic operations and variables
- **float_ops.zen** - Floating point operations

## Advanced Examples

- **functions.zen** - Function definitions and calls
- **nested_calls.zen** - Nested function calls (demonstrates recent bug fix)
- **control_flow.zen** - If/else statements and while loops
- **multiline.zen** - Multi-line program structure

## Running Examples

```bash
# Compile and run
zen run examples/hello.zen

# Or compile first, then run
zen compile examples/hello.zen
./hello
```

## Notes

- All examples are tested and working with Zen v0.0.1
- Examples demonstrate core language features
- More examples will be added as the language evolves
